Importar Nuevos Productos

Este Excel agrega productos nuevos a la base de datos.

Columnas obligatorias: NOMBRE, CATEGORIA

Columnas opcionales: SUBCATEGORIA, COSTO, PORCENTAJE, STOCK, DESCRIPCION

El precio se calcula: COSTO x (1 + PORCENTAJE/100). Si no hay costo, el precio queda en $0.
